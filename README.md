# LA BRISA BUS

This repository contains the first final project completed during Semester Three in 2024. It is a basic website project built as my first website.

## Project Overview

- Project name: **LA BRISA BUS**
- Completed as my first website project during Semester Two, 2024
- A bus ticketing website built with HTML and static assets
- Designed to showcase a simple ticket booking flow and travel information for multiple routes

## Features

- Homepage with navigation to ticket booking and route information
- Multiple catalog pages for different services and classes
- Booking pages for routes such as Kuala Lumpur, Melaka, Cameron Highlands and more
- Payment pages for different ticket types and services
- Static informational pages with route details and travel options
- Includes a `picture/` folder for visual assets used by the website

## Files Included

- `HOMEPAGE.html` - main entry page for the website
- `catalogue_ct.html`, `catalogue_mr.html`, `catalogue_ms.html`, `catalogue_plus.html` - service catalogs
- `payment_ct.html`, `payment_mr.html`, `payment_ms.html`, `payment_plus.html` - payment pages
- `check_booking.html` - booking status or verification page
- Route-specific pages like `kl_ch.html`, `kl_mel.html`, `kl_ter.html`, `mel_ch.html`, `mel_ter.html`, `ter_ch.html`
- `mayangsari.html`, `cepat.html`, `plus.html` - service and route landing pages
- SQL text files for database creation examples (not active code)

## How to Use

1. Open the project folder in your browser or code editor.
2. Launch `HOMEPAGE.html` to start the website.
3. Navigate through the catalog and booking pages to explore the design.

## Notes

This project is intended for upload to GitHub as a class project showcase and portfolio example. It demonstrates an early website built during a school semester using static HTML pages.
